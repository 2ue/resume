> 一个简单的个人经历展示web网址模板

## 项目说明
基于vue-cli（vue2）构建的前端静态项目，可以快速生成个人的简历web网站；

## 作品类型
web

## 功能模块
主要包含以下模块，可以在源码中修改数据
- 个人基本信息
- 个人过往经历
- 个人技能
- Github仓库信息展示（基于github开发接口拉取数据）
